Web-based media visualization experiments

Dataset

Title: Album Covers released in 2018

Description: Wikipedia records 315 albums released in 2018. The images were downloaded from albumoftheyear.org

Total Files: 315

Format: JPG

Resolution: 690 x 690 pixels

For each file, we extracted brightness, saturation, and hue, summarizing in median and standard deviation.

Extraction was made using with ImageJ, using custom script image-measure-HSB.txt and image-measure-RGB.txt

Data: myData.json
