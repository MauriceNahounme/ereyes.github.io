Gabarits HTML pour collections et items Dublin Core

Démo par type de média

image jpg ou png vidéo mp4, mov ou avi audio mp3 texte pdf, txt ou xml

Utilisation

Placez les styles suivants dans la balise <head> 

      
        <head>
          <link rel="stylesheet" href="http://ereyes.github.io/DC_gabarits/item.css">
          <link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">
        </head>

Ensuite, placez aussi les scripts suivants, toujours dans la balise <head> 

      
        <script type="text/javascript" src="https://code.jquery.com/jquery-3.1.1.min.js"> </script>
        <script type="text/javascript" src="http://ereyes.github.io/DC_gabarits/script.js"> </script>

Dans la balise body, placez les éléments suivants :

      
        <body>
          <!-- Ce bloque suivant est la boîte d'aperçu de l'élément -->
          <div class="media" id="preview">
            <p>Aperçu :</p>
          </div>

          <!-- Ce bloque indique les éléments HTML dans lesquelles s'affichent les informations des balises <meta> -->
          <div class="infos">
            <div id="title" class="categorie"><span>Titre de l'item : </span></div>
            <div id="creator" class="categorie"><span>Auteurs :</span></div>
            <div id="sujet" class="categorie"><span>Sujet : </span></div>
            <div id="description" class="categorie"><span>Déscription : </span></div>
            <div id="publisher" class="categorie"><span>Source d'information : </span></div>
            <div id="contributor" class="categorie"><span>Contributor : </span></div>
            <div id="date" class="categorie"><span>Date : </span></div>
            <div id="type" class="categorie"><span>Type : </span></div>
            <div id="format" class="categorie"><span>Format : </span></div>
            <div id="identifier" class="categorie"><span>Identifier : </span></div>
            <div id="source" class="categorie"><span>Source : </span></div>
            <div id="language" class="categorie"><span>Language : </span></div>
            <div id="relation" class="categorie"><span>Relation : </span></div>
            <div id="coverage" class="categorie"><span>Coverage : </span></div>
            <div id="rights" class="categorie"><span>Rights : </span></div>
          </div>
        </body>

Maintenant, vous pouvez placer les balises Dublin Core dans la partie <head> : <meta name="DC.Title" content="...">
